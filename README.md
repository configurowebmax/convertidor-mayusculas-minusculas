# Convertidor de Mayúsculas y Minúsculas Gratis

Aplicación web gratuita para **convertir texto** a diferentes formatos: MAYÚSCULAS, minúsculas, Tipo Título u Oración.

## ✨ Características

- Conversión a **MAYÚSCULAS**, **minúsculas**.
- Formato **Tipo Título** y **Tipo Oración**.
- Contador de **caracteres, palabras y líneas**.
- Copiar texto con un clic.

## 🚀 Uso

1. Escribe o pega tu texto.
2. Elige el formato de conversión.
